# PV PIM-III

Plataforma web para realização de avaliações, permitindo que alunos respondam avaliações, professores criem e corrijam provas, e administradores e moderadores gerenciem o sistema.

## Sobre o Projeto

Esta plataforma tem como objetivo centralizar o processo de avaliação acadêmica de forma digital, oferecendo interfaces dedicadas para cada perfil de usuário: Aluno, Professor, Administrador e Moderador.

## Funcionalidades

- Realização de avaliações online pelos alunos
- Criação e gerenciamento de avaliações pelos professores
- Correções e devolutivas de avaliações
- Sistema de SAQ (Solicitações e Atendimentos) e FAQ
- Gerenciamento de credenciais e monitoramento
- Moderação de tickets e respostas a dúvidas frequentes
